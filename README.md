# FightFit Store - Site statique HTML/CSS

Un site vitrine pour un magasin de vêtements de sports de combat (MMA, Boxe, Muay Thai, JJB).

## Structure

- 5 pages HTML: `index.html`, `products.html`, `disciplines.html`, `about.html`, `contact.html`
- CSS: `css/style.css` (base), `css/responsive.css` (media queries)
- Dossier images: `images/`

## Points pédagogiques couverts

- Balises sémantiques: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`
- Navigation commune sur toutes les pages avec mise en évidence de la page active
- Flexbox: en-tête, hero, formulaires
- CSS Grid: grilles de produits, sections d'information
- Formulaire complet et accessible (labels, `required`, placeholders)
- Responsive design: mobile (<768px), tablette (768–1024px), desktop (>1024px) via media queries
